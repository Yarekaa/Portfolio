ABOUT

Web programming is formed from 2 words, namely programming and web where programming itself is the process or method of executing a sequence of instructions or commands given to the computer to create certain functions or tasks. and the Web is a system for accessing, manipulating, and downloading documents contained on a computer that is connected via the internet or a network.

So Web Programming is a process or a way to run instructions on a computer that is connected to the internet to make certain functions or tasks.. and in running a program on the web through a web browser such as: mozilla, Opera, Chrome, etc.
